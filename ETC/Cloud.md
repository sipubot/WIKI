# 클라우드 및 분산 시스템 (Cloud and Distributed Systems)

## 개요
현대적 인프라 구축 능력을 강화하기 위해 클라우드 플랫폼 경험을 추가하세요.

## 주요 클라우드 플랫폼

### AWS (Amazon Web Services)
- **컴퓨팅**: EC2, Lambda, Fargate
- **스토리지**: S3, EBS, RDS
- **네트워킹**: VPC, CloudFront, Route 53
- **관리 도구**: CloudFormation, CloudWatch

### Azure
- **컴퓨팅**: Virtual Machines, Functions, Kubernetes Service
- **데이터베이스**: Cosmos DB, SQL Database
- **AI/ML**: Machine Learning, Cognitive Services

### GCP (Google Cloud Platform)
- **컴퓨팅**: Compute Engine, Cloud Run, Kubernetes Engine
- **빅데이터**: BigQuery, Dataflow
- **AI**: AI Platform, AutoML

## 분산 시스템 개념

### 마이크로서비스 아키텍처
- **서비스 분해**: 모놀리식에서 마이크로 서비스로 전환
- **API 게이트웨이**: 서비스 간 통신 관리
- **서비스 메시**: Istio, Linkerd

### 컨테이너 오케스트레이션
- **Kubernetes**: 컨테이너 배포 및 관리
- **Docker Swarm**: 경량 오케스트레이션
- **Helm**: Kubernetes 패키지 매니저

### 이벤트 기반 아키텍처
- **메시지 큐**: Kafka, RabbitMQ
- **이벤트 스트리밍**: 실시간 데이터 처리
- **CQRS 패턴**: 명령과 쿼리 분리

## 실무 프로젝트 제안

1. **클라우드 네이티브 애플리케이션**
   - 서버리스 함수 배포 (Lambda/Azure Functions)
   - 컨테이너화된 앱 Kubernetes 배포

2. **분산 데이터베이스 구축**
   - 다중 리전 데이터베이스 설정
   - 데이터 복제 및 샤딩 전략

3. **모니터링 및 로깅 시스템**
   - CloudWatch/Azure Monitor 활용
   - 분산 추적 구현 (Jaeger, Zipkin)

4. **CI/CD 파이프라인**
   - GitHub Actions/Azure DevOps
   - 인프라 코드 (Terraform/CloudFormation)
